# ListLens (Android)

Snap a book (ISBN-first) → confirm metadata → capture photos → create an eBay draft listing.

## Status
Early scaffold.

## MVP scope (phase 1)
- Books category
- Camera ISBN scan (barcode first, OCR fallback)
- Metadata lookup (ISBN-based)
- Confirm/retake/cancel
- Capture up to 5 photos total
- (Later) eBay OAuth + create draft listing
